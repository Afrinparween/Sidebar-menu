#📸 Photography Sidebar Menu – HTML & CSS Mini Project

This is a simple and clean Photography-themed Sidebar Menu UI built using pure HTML and CSS. The project demonstrates how a responsive and interactive navigation menu can be created without using JavaScript, only with CSS techniques.

The layout features a full-screen background image with a slide-in sidebar menu, commonly used in modern websites and portfolios.

🚀 Features

📷 Full-screen photography background layout

📑 Sidebar navigation menu

🍔 Hamburger menu toggle

✨ Smooth CSS transitions and hover effects

🎨 Modern UI design with shadows and transparency

🔤 Google Fonts integration (Poppins)

⭐ Font Awesome icons for navigation and social media

🛠️ Technologies Used

HTML5 – Page structure

CSS3 – Styling and animations

Font Awesome – Icons

Google Fonts – Typography
